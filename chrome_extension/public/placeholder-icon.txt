# Extension Icons Placeholder

You need to create icon files:
- icon16.png (16x16px)
- icon48.png (48x48px)
- icon128.png (128x128px)

## Quick Icon Creation

### Option 1: Use Figma/Canva
1. Create a square canvas (128x128px)
2. Design your icon (suggested: magnifying glass with check mark)
3. Export as PNG in all three sizes

### Option 2: Use ImageMagick (command line)
```bash
# If you have a source icon (icon.png)
convert icon.png -resize 16x16 icon16.png
convert icon.png -resize 48x48 icon48.png
convert icon.png -resize 128x128 icon128.png
```

### Option 3: Use Online Generator
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

### Temporary Solution
For development, you can use simple placeholder colors:
1. Create solid color squares in any image editor
2. Name them icon16.png, icon48.png, icon128.png
3. Place in the public/ folder

The extension will work without icons, but Chrome will show a default puzzle piece icon.
